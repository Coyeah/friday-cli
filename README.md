# oops-cli

快速生成一个项目脚手架。使用方式单一且简单。

## 安装

```shell
# npm install oops-cli -g
```

## 使用

构建项目，react & antd & mobx & react-router;

```shell
# oops project projectname
```

构建简单应用，react & antd;

```shell
# oops project projectname
```
